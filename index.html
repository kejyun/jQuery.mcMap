jQuery mcMap (jQuery Manage Control Map)

版本(Version)

v 1.0

修改日期(Last modify)

2013/03/30

作者(Author)

KeJyun Hong

描述(Descrition)

動態管理在Google Map API地圖上的繪圖資訊

修改紀錄(Change log)

加入功能

  • 路徑顯示控制(MapContainer.ToggleRoute(routeKey))
  • 路徑標記顯示控制(MapContainer.ToggleRouteMarker(routeKey))
  • 折線顯示控制(MapContainer.TogglePolyline(polylineKey))
  • 標記顯示控制(MapContainer.ToggleMarker(markerKey))
  • 新增折線(MapContainer.AddPolyline(option))
  • 新增標記(MapContainer.AddMarker(option))
  • 新增路徑(MapContainer.AddRoute(option))

引用Google Map API及jQuery

參數(Option)

名稱 類型 說明
onAfter function 後處理函式
onBefore function 前處理函式
center object 中心點位置 
center: {
    x: '25.040893', 
    y: '121.515827'
},
zoom integer 地圖縮放大小

範圍:0~18

draggable ool 是否可以拖曳
mapTypeId string 地圖類型

ROADMAP、SATELLITE、HYBRID、TERRAIN，預設ROADMAP

mapTypeControl bool 是否開啟地圖類型控制
polyline array 繪製折線 
[
    {
      coords: [         // 折線點

      width: 5,         // 線寬(預設2)
